## 🚀 快速开始

```
双击 index.html       # 电脑端
手机浏览器直接打开     # 自动适配触屏
```

无需安装、无需服务器，浏览器打开即玩。

## 🎮 操作说明

| 操作 | 方式 |
|------|------|
| 选择关卡 | 点击关卡卡片 |

## ⚙️ V4 技术改进

### Delta Time 帧率稳定
- 传统帧计数法在不同刷新率下会导致速度不一致
- V4 引入基于时间的游戏循环，确保 60Hz 和 144Hz 屏幕游戏速度一致
- 游戏速度不再受帧率波动影响

### 📱 手机触屏适配
- 新增 `touchstart` / `touchmove` / `touchend` 事件监听
- 触摸事件阻止页面滚动，游戏内流畅操作
- CSS 媒体查询自动缩放画布，适配各种屏幕宽度
- 覆盖层按钮使用视图宽度的相对单位（vw），自动适应屏幕
- 选关 / 选植物 / 放置 / 收集阳光全部支持手指触控

### 选关左右拉条
- 关卡数已增至 **5 关**，一屏显示不下时自动启用翻页
- 点击 ◀ ▶ 箭头或按键盘 `←` `→` 翻页
- 底部**指示点**实时显示当前位置
- 每次显示 3 张关卡卡片

### 冷却数字显示
- 植物卡片冷却时显示剩余帧数，更直观
- 玩家可以精确知道何时可以再次使用该植物

### 代码去重
- Plant 类的 `draw()` 方法大量重复代码已合并抽象
- 僵尸渲染逻辑也做了简化
