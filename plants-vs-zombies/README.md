## ⚙️ 游戏机制

### 阳光经济
- **初始阳光**：200
- **天空阳光**：每 5 秒自动掉落一颗（25 阳光）
- **向日葵产出**：每约 4.7 秒产生一颗（25 阳光）
- **波次奖励**：每波清完奖励 50 阳光
- **铲除返还**：铲除植物返还费用的 50%
- **自动收集**：阳光落地后 3 秒未点击自动收集

### 波次系统
- 第 1 波：4 只僵尸，生成间隔 5 秒
- 每波增加 1 只僵尸，波间休息 6 秒
- 僵尸类型随波次逐渐多样化
- 第 3 波开始可能出现铁桶僵尸

### 割草机
- 每行左侧有一台割草机
- 僵尸接近左侧红线时自动启动
- 碾压该行所有僵尸后消失

### 胜负条件
- 🎉 **胜利**：暂无（无尽模式，挑战更高波次）
- 💀 **失败**：任意僵尸到达左侧红线

## 🎨 技术实现

- **渲染**：HTML5 Canvas 2D
- **游戏循环**：`requestAnimationFrame` 60fps
- **碰撞检测**：基于坐标距离
- **粒子系统**：爆炸、击中、铲除等特效
- **波次管理**：渐进式难度递增

## 🔧 自定义调整

如果你想调整游戏难度，可以修改 `index.html` 中的以下参数：

```javascript
// ── 游戏状态（约第 100 行）──
let sun = 200;              // 初始阳光

// ── 僵尸速度（约第 448 行）──
this.speed = 0.17;          // 移动速度（越大越快）

// ── 波次公式（约第 1177 行）──
totalZombiesInWave = 3 + wave;  // 每波僵尸数
spawnInterval = Math.max(150, 300 - wave * 8); // 生成间隔(帧)

// ── 阳光掉落间隔（约第 1143 行）──
if (skySunTimer > 300) { ... }  // 天空阳光间隔(帧)

// ── 自动收集延迟（Sun 类 update 方法）──
if (this.idleFrames >= 180) { ... }  // 3秒=180帧
```
