# 🌻 植物大战僵尸 V2 — 关卡版

纯 HTML5 Canvas + JavaScript，单文件即玩。V2 在 V1 基础上新增**关卡选择系统**、**僵尸博士 Boss 战**和 **2 种新植物**。

## 🚀 快速开始

```
双击 index_v2.html
```

无需安装、无需服务器，浏览器打开即玩。

---

## 🗺️ 关卡系统

启动后进入关卡选择画面，3 张卡片可选：

| # | 关卡名 | 图标 | 波次 | 初始阳光 | 可用僵尸 | Boss |
|---|--------|------|------|----------|----------|------|
| 1 | 前院草地 | 🌞 | 5 | 200☀ | 普通 | — |
| 2 | 夜幕降临 | 🌙 | 7 | 150☀ | 普通 / 路障 / 旗手 | — |
| 3 | 终极对决 | 💀 | 8 | 250☀ | 全部类型 | 第 8 波：僵尸博士 |

- 每波僵尸数 = 关卡基础数 + (波次-1) × 增量
- 波间休息 5~6 秒，清完一波奖励 50 阳光
- 过关 / 失败后可返回选关或重试

## 💀 僵尸博士 Boss 战

关卡 3 第 8 波登场，位于中间行。

| 属性 | 数值 |
|------|------|
| 血量 | **4000 HP** |
| 移动速度 | 0.07（极慢，约为普通僵尸的 40%） |
| 体型 | 普通僵尸的 **2.2 倍** |
| 啃食伤害 | 3.0（普通僵尸的 3 倍，快速摧毁植物） |
| 技能 | 每 **~6 秒召唤 1 只普通僵尸**（场上 <15 只时） |

### 外观特征
- 灰色机械装甲身躯 + 红色管线
- 暴露的粉色大脑
- 血红双眼 + 金色瞳孔
- 机械爪子双臂
- 头顶显示血条 + 血量数值
- 召唤前头顶闪烁 `⚠ 召唤中...`

### 战斗策略
1. 前排布置大量坚果墙拖延
2. 后排密集双发射手 + 寒冰射手输出
3. 土豆地雷埋在博士必经之路上
4. 樱桃炸弹清理博士召唤的小僵尸群
5. 减速效果对博士同样有效

## 🎮 操作说明

| 操作 | 方式 |
|------|------|
| 选择植物 | 点击卡片，或键盘 `1`~`7` |
| 放置植物 | 选中后点击草坪格子 |
| 收集阳光 | 点击阳光 / 3 秒自动收 |
| 铲除植物 | 点击 🔧 → 点击植物（返还 50% 阳光） |
| 取消选择 | `ESC` / 右键 |
| 返回选关 | 游戏结束后点击按钮 |

## ⚙️ 游戏机制

### 阳光经济
- 初始阳光：关卡 1/2/3 分别为 200/150/250
- 天空阳光：每 5 秒自动掉落
- 向日葵产出：每 ~4.7 秒一颗
- 波次奖励：每波清完 +50
- 铲除返还：费用的 50%
- 自动收集：落地 3 秒未点击自动收

### 割草机
每行左侧一台，僵尸接近时自动启动碾压。对 Boss 造成 500 伤害并减速。

## 🔧 自定义修改

在 `index_v2.html` 中搜索对应参数：

```javascript
// ── 关卡配置（约第 110 行）──
LEVELS = [
  { waves: 5, startSun: 200, ... },  // 关卡 1
  { waves: 7, startSun: 150, ... },  // 关卡 2
  { waves: 8, startSun: 250, bossWave: 8, ... }, // 关卡 3
]

// ── 植物数据（约第 160 行）──
repeater: { cost: 200, cooldown: 90, ... }
potato_mine: { cost: 25, cooldown: 500, ... }

// ── Boss 属性（约第 370 行）──
case 'boss':
  this.hp = 4000; this.speed = 0.07; ...

// ── 土豆地雷准备时间（Plant 构造函数）──
this.armDuration = 750;  // 750帧 = 12.5秒
```
