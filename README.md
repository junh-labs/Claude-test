# Claude Code 实验场 🧪

使用 [Claude Code](https://claude.ai/code) AI 编程助手构建的各类小项目集合。

## 📂 项目列表

| 项目 | 说明 |
|------|------|
| 🌻 [`plants-vs-zombies/`](plants-vs-zombies/) | **植物大战僵尸** — 纯 HTML5 Canvas 网页游戏，从 V1 迭代到 V4 |
| 🧪 `test1/` | 功能测试项目 |
| 🧪 `test2/` | 功能测试项目 |

## 🚀 快速体验

```
cd plants-vs-zombies/v4
双击 index.html
```

无需安装，浏览器即可运行。

## 🎯 目的

本仓库用于探索和验证 Claude Code 在以下维度的能力：
- **全栈开发**：从零构建完整游戏
- **版本迭代**：逐步增加功能，重构优化
- **文件组织**：项目结构整理与文档维护
- **问题修复**：Bug 修复与性能优化

## 📝 启动脚本

| 脚本 | 用途 |
|------|------|
| `start-DS-cc.ps1` | 启动 Claude Code（DeepSeekV4Pro 模式） |
| `start-DSF-cc.ps1` | 启动 Claude Code（DeepSeekV4flash 快速模式） |
